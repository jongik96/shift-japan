# Public Assets

이 폴더에 정적 파일을 저장하세요.

## Favicon (아이콘)

다음 파일들을 추가하세요:
- `favicon.ico` - 기본 파비콘 (32x32, 16x16 포함)
- `icon.png` 또는 `apple-icon.png` - 180x180 크기 (Apple 기기용)
- `icon.svg` - 벡터 아이콘 (선택사항)

## OG (Open Graph) 이미지

다음 파일들을 추가하세요:
- `og-image.png` 또는 `og-image.jpg` - 1200x630 크기 (SNS 공유용)

## 이미지 최적화 사이트

- Favicon 생성: https://favicon.io/
- OG 이미지 체크: https://www.opengraph.xyz/

## Next.js 사용

Next.js는 자동으로 이 폴더의 파일들을 서빙합니다.
예: `public/favicon.ico` → `/favicon.ico`
